# Project Frontier Design Bible

## Visual identity
- Threshold: blush pink, ivory, antique gold, celestial manuscript.
- Mission Control: midnight blue, black, steel, cyan, amber.
- Codex: parchment, leather, wax seals, ink.
- Character: dark neutral, gold, discipline accents.
- Sanctuary: forest green, wood, warm natural light.
- Observatory: stars, nebulae, violet, brass instruments.
- Atlas: maps, compass, brass, constellation routes.
- Archive: walnut, parchment, library, artifacts.

## Navigation
Desktop uses a permanent left sidebar. Mobile uses a drawer plus a five-item bottom navigation bar.

## Communication
The Game Master speaks calmly and precisely. It reports facts, uncertainty, confidence, and missing information without excessive praise or pressure.

## Current vertical slice
Threshold → Mission Control → Daily Quests → Captain's Log → Archive.
